Diagramming tool: draw.io

Posted on Sep 12, 2018 by Klaus Paul 3 mins read

In my work laptop, running Windows 7 Enterprise, I have Visio Standard installed. I’ve grown accustomed to a couple of features that make for a productive experience, for example:

  • Shift+mouse wheel, to move sideways
  • Control+mouse wheel, to zoom in and out
  • The alignment guides
  • Layers, to easily create complex diagrams and also easily select elements
  • A vast library of stencils
  • Ability to create your own stencils
  • Ability to export to VSDX and VDX (because I can import those into Gliffy and have my diagram shared easily into a Confluence page)

As many of you may know, there is no Visio for MacOS (at least at the time I wrote this). This thread is from 2017, and the official Microsoft moderator states that “Based on my search, there is no ETA when Visio will be released in Mac OS”.

My home laptop is a Mac Air 2012. A couple of time, for personal reasons, I had the need to either draw a flowchart or mock-up a website page, but not having a Mac version of Visio, and having just an occasional need for diagrams, made me search for alternatives.

I’ve tried a couple of tools, such as Pencil Project or Dia, but years using Visio created some expectations. I then tried cloud solutions such as Lucid Chart and Creately, but I didn’t like the idea of having to be connected to the internet to be able to use them (in addition to having my diagrams in the cloud, somewhere in the world).

I found out about draw.io, but initially, it appeared to have the same problem that Lucid Chart and Creately both have: it was a cloud solution. Or so I thought.

Buried deep within that website there are options to download a free, open-source version of draw.io that allows for its use while connected to the internet. Or, if you are not as dumb as I am, you can just click on the button below, which appears when using draw.io on the web:

How come I didn’t see this before?

One can create as many diagrams as one might ever want or need, for free, for any purpose. And those diagrams are stored in your local computer. Or if one wants, in a sharing platform as iCloud Drive, Google Drive, Dropbox, Box, or any other.

If you have used Visio before, the learning curve is smooth. Menu items are comparable, the selection of shapes is also drag-n-drop to the canvas, connection points, drawing of connections, and so on. There is a good number of shape options and their design is pleasant. One thing I didn’t see in Visio: an option to auto-save the diagram. If you are like me, you won’t need to be hitting Ctrl+S that often when using the tool.

To me, draw.io is a solid alternative to paid products, and to anyone who needs to do occasional diagrams, but want good features, performance, and easy of use.

The only two points that I dislike about the tool:

  1. I couldn’t find a way to create my own shapes. Once you finds out how to do it, this is a simple task in Visio.
  2. It seems the installation sets the flag “Comic” as checked. What does that mean? Look at the image below. Did you notice the line connecting the boxes is wiggling? The next image shows why that is. Once the offender checkbox is uncheck, all connections lines in that diagram will be straight. However, start a new diagram and you will need to repeat the procedure.
